## Instalação

```
npm install
```

## Inicialização

```
npm start
```
  

## Testes

  

```sh
npm run test
```

## Documentação

Esta API utiliza a documentação interativa do Swagger para testes manuais. Acesse a rota a seguir para interagir com a API:

```
http://localhost:4000/docs/
```
